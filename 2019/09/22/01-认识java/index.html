认识Java

Author: CodePaul Date: 2019-09-22  17:00:48

Java基础

注释

​ 什么是注释？为什么要学注释？

​ 先来回答第一个问题。什么是注释？注释是对代码的解释和说明，目的让我们更加轻松理解代码。正确的注释不会影响为代码的编译和运行。

​ 那么为什么要学注释。无论是在团队开发还是在学习官方api文档的过程中注释都十分的重要。在程序开发过程中，你的代码不止你一个人阅读，还需要被团队的其他成员阅读，那其他人又怎么自动你的代码是用来做说明功能的呢？所以你就需要为你的代码加以解释，这样他们就能更好的理解你的代码（可能你自己写的代码，过了几天你就不知道这串代码是写来干什么的），所以添加注释是十分必要的，十分必要的，十分必要的

Java中有单行注释，多行注释，文档注释。

  • 单行注释 以//开头 以换行结尾

  • 多行注释 以/*开头,以*/结尾

  • 文档注释 以/**，以*/结尾

关键字

关键字：是指在程序中，Java已经定义好的单词，具有特殊含义。

  • 例如有 public 、 class 、 static 、 void 等，这些单词已经被Java定义好

  • 关键字的特点：全部都是小写字母 。

  • 关键字比较多，不需要死记硬背，在后面的文章里面会有说明。

    注意：Java不是关键字（Java才是正确的写法，java，JAVA都是错误写法）

常量

常量：是指在程序的执行过程中，其值不可以发生改变的值。

常量的分类：

  • 自定义常量：通过final关键字定义（后面的文章会说明）。

  • 字面值常量：

    字面值常量 举例
    字符串常量 “HelloWorld”
    整数常量 233
    浮点常量 3.14
    字符常量 ‘a’, ‘b’
    布尔常量 true, false
    空常量 null

    注意null 是空常量。不能直接在输入语句中打印 System.out.println(null);

    字符常量单引号里面只能有一个字符。

变量

变量：在程序执行的过程中，其值可以发生改变的量

变量定义的格式包括三个要素： 数据类型 、 变量名 、 数据值 。

数据类型

Java的数据类型分为两大类：

  • 基本数据类型 ： 包括 整数型， 浮点型， 字符型，布尔型。
  • 引用数据类型：类，数组，接口。
基本数据类型分类

Java中的默认类型：整数型默认是int，浮点型是double。

定义long类型，要在数据值后面加L。

定义float类型，要在数据值后面加F。

引用数据类型会在后面的文章中讲。

标识符

标识符：是指在程序中，我们自己定义内容。比如变量的名字，类的名字、方法的名字和等等，都是标识符。

标识符命名规则：

  • 标识符可以包含 英文字母26个(区分大小写) 、 0-9数字 、 $（美元符号） 和 _（下划线） 等。

  • 标识符不能以数字开头。

  • 标识符不能是关键字。

标识符命名规范：

  • 变量名规范

    • 一个单词，字母全部小写
    • 多个单词，一个单词首字母小写，第二个单词以后（包含）每个单词首字母大写（小驼峰式）

  • 类名规范：

    • 一个英文单词，首字母大写，例如：Dog
    • 多个英文单词，每个单词首字母大写（大驼峰式），例如：HelloWorld

  • 方法名规范：

    • 一个单词，字母全部小写，例如：main
    • 多个单词，一个单词首字母小写，第二个单词以后（包含）每个单词首字母大写（小驼峰式）；例如：getMax，getSum，getName

变量命名注意事项

  • 如果创建多个变量，那么变量之间的名称不可以重复。
  • 对于float和long类型来说，字母后缀F和L不要丢掉。
  • 如果使用byte或者short类型的变量，那么右侧的数据值不能超过左侧类型的范围。
  • 没有进行赋值的变量，不能直接使用；一定要赋值之后，才能使用。
  • 变量使用不能超过作用域的范围。 【作用域】：从定义变量的一行开始，一直到直接所属的大括号结束为止。
  • 可以通过一个语句来创建多个变量，但是一般情况不推荐这么写。
